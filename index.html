Reminder(이하 "앱")는 나중에 볼 링크·메모를 저장하고 지정 시간에 알림으로 상기시켜 주는 서비스입니다. 본 방침은 앱이 수집하는 정보와 그 사용 방식을 설명합니다.

1. 수집하는 정보

  • 계정 정보: Google 로그인 시 이메일 주소 및 프로필 이름 (비로그인 사용 시 익명 식별자)
  • 저장 항목: 사용자가 직접 입력하는 링크, 텍스트, 메모
  • 푸시 알림 토큰: 리마인드 알림 발송에 사용되는 기기 식별자
  • 설정 정보: 알림 시간, 시간대(timezone), 언어 설정

2. 정보 사용 목적

  • 저장된 항목을 지정 시간에 알림으로 전달
  • 계정 식별 및 데이터 동기화
  • 서비스 개선 및 오류 분석

3. 제3자 서비스

  • Supabase: 데이터베이스 및 인증 (개인정보처리방침)
  • Expo: 푸시 알림 전송 (개인정보처리방침)
  • Google: OAuth 로그인 (개인정보처리방침)

4. 데이터 보안

수집된 데이터는 Supabase의 암호화된 서버에 저장되며, Row Level Security(RLS)를 통해 본인의 데이터에만 접근할 수 있도록 보호됩니다.
